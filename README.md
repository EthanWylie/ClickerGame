# ClickerGame

The primary objective is to click a button, which will increase your score.

## Features

* Clicking A Button
* Music
* Shop Upgrades

## Development Tools

ClickerGame was built with Java, using the Swing library

## Installation and Usage
To install ClickerGame:

1. Have Java installed.
2. Clone this repository.
3. Compile the Java files.
4. Run ClickerGame.
